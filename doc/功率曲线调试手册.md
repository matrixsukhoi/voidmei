# VoidMei 功率曲线调试手册

> **作者**: VoidMei 核心贡献者
> **目标读者**: 需要验证或调试活塞发动机功率计算的开发者

## 1. 概述

VoidMei 的活塞发动机功率计算基于 WAPC (wt-aircraft-performance-calculator) 的算法实现。
当功率曲线出现异常时，可以使用本手册中的方法进行对比调试。

### 工作原理

```mermaid

    A[FM 文件 .blkx] --> B[Blkx.java 解析]

    E --> F[功率值 hp]

    subgraph "参数提取"

    C
    end

    subgraph "功率计算"

    E
    end
```

## 2. 调试脚本

我们提供了两个对比调试脚本，用于验证 VoidMei 与 WAPC 的计算结果是否一致。

### 2.1 VoidMei 侧脚本

**文件**: `test/DebugYak3PowerCurve.java`

**功能**: 硬编码 FM 参数，调用 `PistonPowerModel` 计算功率，输出详细日志。

**运行方式**:
```bash
# 编译
javac -encoding UTF-8 -d bin -classpath bin test/DebugYak3PowerCurve.java

# 运行（启用 DEBUG 日志级别）
java -classpath bin DebugYak3PowerCurve > /tmp/voidmei_power.log
```

### 2.2 WAPC 侧脚本

**文件**: `script/debug_wapc_yak3.py`

**功能**: 相同的硬编码参数，调用 WAPC Python 函数计算功率。

**运行方式**:
```bash
# 需要 WAPC 项目路径
cd /path/to/wtapc
python3 script/debug_wapc_yak3.py > /tmp/wapc_power.log
```

### 2.3 对比结果

```bash
diff /tmp/voidmei_power.log /tmp/wapc_power.log
```

## 3. 日志输出格式

两个脚本输出统一格式，便于 diff 对比：

```
=== PARAMETER EXTRACTION ===

--- Stage 0 Parameters ---
  critAlt=300, critPower=1310
  deckAlt=0, deckPower=1290
  ceilingAlt=5000, ceilingPower=670
  ...

=== POWER CURVE (military, static, stage 0) ===

[alt=0] ramAlt=0
  variabler: lower=(0.0, 1290.0), higher=(300.0, 1310.0), curv=1.0
  equationer: power=1290.0

[alt=300] ramAlt=300
  variabler: lower=(0.0, 1290.0), higher=(300.0, 1310.0), curv=1.0
  equationer: power=1310.0
...
```

### 关键字段说明

| 字段 | 说明 |
|------|------|
| `critAlt` | 临界高度 (m) - 增压器能维持最大增压的最高高度 |
| `critPower` | 临界功率 (hp) - 在临界高度的功率 |
| `deckPower` | 海平面功率 (hp) - 低于临界高度时的功率 |
| `ceilingAlt` | 升限高度 (m) - 功率曲线的参考终点 |
| `ceilingPower` | 升限功率 (hp) - 升限高度的功率 |
| `ramAlt` | RAM 有效高度 - 考虑动压后的等效高度 |
| `lower/higher` | 插值边界 - (高度, 功率) 元组 |
| `curv` | 曲率系数 - 插值曲线的形状参数 |

## 5. 参数提取链路

当怀疑参数提取有问题时，检查以下链路：

### 5.1 Blkx.java 提取

```java
// 关键参数位置 (Blkx.java:758-768)
compAlt[i] = getdouble("Compressor.Altitude" + i);
compPower[i] = getdouble("Compressor.Power" + i);
compCeil[i] = getdouble("Compressor.Ceiling" + i);
compCeilPwr[i] = getdouble("Compressor.PowerAtCeiling" + i);
compRpmRatio[i] = getdouble("Compressor.PowerConstRPMCurvature" + i);
```

### 5.2 FMPowerExtractor 转换

```java
// 关键转换逻辑 (FMPowerExtractor.java)
// Pass 1: 基础参数映射
stages[i].critAlt = blkx.compAlt[i];
stages[i].critPower = blkx.compPower[i];

// Pass 2: RPM 调整 (definition_alt_power_adjuster)
if (needsRpmAdjustment) {
    adjustPowerAndAltitude(stages[i], blkx, i, defaultRpm, stageDeckPower);
}

// Pass 3: WEP 参数计算
stages[i].wepPowerMult = calculateWepMultiplier(blkx, i);
stages[i].wepCritAlt = calculateWepCriticalAltitude(blkx, stages[i], i);
```

## 6. 常见问题排查

### Q: 静态功率 (speed=0) 不匹配

**检查点**:
1. `deckPower` 是否正确 (Stage 0 = `Main.Power`, Stage 1+ = 0.8 × 上一级)
2. `critAlt` 和 `critPower` 是否正确读取
3. `curvature` (PowerConstRPMCurvature) 是否为正确值

### Q: RAM 效果下功率不匹配

**检查点**:
1. `speedManifoldMult` (SpeedManifoldMultiplier) 是否正确
2. RAM 有效高度计算：`AtmosphereModel.ramEffectAltitude()`
3. VoidMei 使用 `double`，WAPC 使用 `int()` 截断（通常不影响结果）

### Q: WEP 功率不匹配

**检查点**:
1. `wepPowerMult` 计算是否正确
2. 当 `wepPowerMult ≈ 1.0` 时应短路返回军用功率
3. `AfterburnerBoostMul` 是否显式设为 0（禁用该级 WEP）

### Q: 多级增压器级别选择不匹配

**检查点**:
1. `optimalPowerAtAltitude()` 应选择功率最大的级别
2. 检查每个级别的 `deckPower` 计算

## 7. 测试用例模板

如需为新飞机创建调试脚本，复制以下模板：

```java
// test/DebugXXXPowerCurve.java
import prog.util.Logger;
import prog.util.PistonPowerModel.CompressorStageParams;
import static prog.util.PistonPowerModel.*;

public class DebugXXXPowerCurve {
    public static void main(String[] args) {
        Logger.setMinLevel(Logger.Level.DEBUG);

        CompressorStageParams stage0 = new CompressorStageParams();
        stage0.critAlt = /* Compressor.Altitude0 */;
        stage0.critPower = /* Compressor.Power0 */;
        stage0.deckPower = /* Main.Power */;
        stage0.ceilingAlt = /* Compressor.Ceiling0 */;
        stage0.ceilingPower = /* Compressor.PowerAtCeiling0 */;
        stage0.curvature = /* Compressor.PowerConstRPMCurvature0, default 1.0 */;
        stage0.exactAltitudes = /* true if ExactAltitudes=true or no CompressorOmegaFactorSq */;
        stage0.speedManifoldMult = /* Compressor.SpeedManifoldMultiplier, default 1.0 */;

        // 测试高度点
        int[] altitudes = {0, 500, 1000, 2000, 3000, 5000, 7000, 10000};

        for (int alt : altitudes) {
            double power = powerAtAltitudeAdvanced(stage0, alt, false, 0, false, 15);
            System.out.printf("[alt=%d] power=%.1f%n", alt, power);
        }
    }
}
```
